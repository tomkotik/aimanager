# Booking Template

Шаблон для быстрого старта агента под бронирование.

Использование:

```bash
agentbox init-booking my-studio --name "My Studio Booking"
```

После создания:
1. Заполните knowledge/*.md
2. Добавьте секреты (`openai_key`, `google_*`, `telegram_bot_token`)
3. Синхронизируйте и запустите агента
